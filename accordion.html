1. What is JavaScript?

JavaScript is a scripting language used to create dynamic web content like interactive forms, animations, and more.

2. What is HTML?

HTML stands for HyperText Markup Language. It structures web content using elements and tags.

3. What is CSS?

CSS stands for Cascading Style Sheets. It controls the layout and appearance of web elements.

4. What is the DOM?

The DOM (Document Object Model) is an interface that allows scripts to dynamically access and update content, structure, and style of a document.

5. What is an API?

An API (Application Programming Interface) allows different software systems to communicate with each other.
